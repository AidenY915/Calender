달력 사용법:
좌우 버튼 클릭 -> 월 변경

특정 날짜 칸을 선택 시 스케쥴 입력 가능.
-> 입력 칸에 아무것도 적지 않거나, 취소를 누르면 스케줄은 추가되지 않는다.

날짜 칸을 선택하면, 달력 아래에 그 날짜의 스케줄 리스트가 표시된다.
리스트에서 수정을 원하는 스케쥴을 클릭하면 수정이 가능하며, 완전히 지우고 확인을 누르면 해당 스케쥴이 삭제가 된다.

달력 우측 하단에 버튼을 누르면 현재 달력의 스케쥴 데이터를 내보내 PC에 저장할 수 있다. 기존에 저장한 스케쥴 데이터를 불러오고 싶으면 바로 아래 버튼을 통해 불러올 수 있다.

새로고침하면 캘린더가 초기화되므로 주의!
